# Task-Tracking-System
TaskMaster: A Collaborative Task Tracking System

## Features
This backend supports:

* User registration and login with JWT authentication
* Role management (Admin/User)
* Create, assign and complete tasks
* **Team/Project collaboration** – users can create or join teams, add members, and associate tasks with teams
* **Comments and attachments** on tasks for rich collaboration
* List tasks by team
* Secure logout endpoint
* Swagger API documentation available at `/api-docs`

Optional future extensions may include real-time notifications and AI-generated task summaries..
